# GOLDBRIX

Public Proof-of-Work Cryptocurrency Network

GOLDBRIX is an open and permissionless blockchain network based on Proof-of-Work.

The network has **no central owner**, **no project keys**, and **no founder allocation**.

Coins enter circulation **only through mining and voluntary market exchange**.

Anyone can run a node.
Anyone can mine.
Anyone can build on top of the network.

---

## Network Principles

• No central authority  
• No project custody  
• No founder allocation  
• No admin keys  
• Open mining for everyone  

GOLDBRIX is a neutral protocol.

Ownership belongs only to the users who mine or acquire coins on the open market.

---

## Mining

Mining is open to anyone running the GOLDBRIX node software.

Example:

generatetoaddress 1 <your_address>


Miners receive block rewards directly to their own wallets.
